# TENS-Electroestimulador-Control
Electroestimulador TENS con control de tren de pulsos y espejo de corriente. Incluye diseño de PCB y reporte IEEE.

Prototipo funcional de un estimulador eléctrico transcutáneo diseñado para fines didácticos.

## Aspectos Técnicos Destacados
* **Espejo de Corriente:** Implementación analógica para asegurar una corriente constante, vital para la seguridad y efectividad del estímulo.
* **Control de Pulsos:** Generación de señales cuadradas/bifásicas con parámetros ajustables de frecuencia y ancho de pulso.
* **Flujo de Diseño:** De la teoría y simulación a prototipo en protoboard y finalmente a **PCB manufacturado**.

## Estructura
* **/docs**: Reporte técnico en formato **IEEE**.
* **/hardware**: Archivos de diseño del PCB y esquemáticos.
* **/media**: Fotos del sistema físico y capturas de osciloscopio.
